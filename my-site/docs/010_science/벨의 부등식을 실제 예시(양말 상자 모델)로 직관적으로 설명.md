
[[벨의 부등식]]
[[우리는 현실 세계에 살고 있는 것이 아니다]]
[[왜 얽힘 상관관계는 빛보다 빠른 통신이 불가능한지]]

## 핵심 요약

- **양말 상자 모델**은 “숨은 변수(이미 정해진 값)” 상황을 설명하는 비유이다.
    
- 상자 두 개에 **왼쪽 양말 / 오른쪽 양말**을 넣어 멀리 보낸다.
    
- 한쪽을 열면 다른 쪽 결과가 **즉시 결정된 것처럼 보인다.**
    
- 그러나 실제로는 **처음부터 값이 정해져 있었던 것**이다.
    
- 벨의 부등식은 **이런 종류의 ‘미리 정해진 값’ 모델이 만들 수 있는 상관관계의 한계**를 계산한 것이다.
    
- 양자 얽힘 실험은 **그 한계를 넘어서는 상관관계**를 보인다.
    
- 그래서 “양말 상자 같은 숨은 변수 설명”이 불가능해진다.
    

---

# 1. 양말 상자 모델 (고전적 직관)

두 상자에 양말을 넣는다.

```
상자 A → 왼쪽 양말
상자 B → 오른쪽 양말
```

멀리 떨어진 두 사람에게 보낸다.

```
Alice : 상자 A
Bob   : 상자 B
```

Alice가 열어서

```
왼쪽 양말
```

을 보면

Bob은 열지 않아도

```
오른쪽 양말
```

일 것을 알 수 있다.

중요한 점

```
결과는 처음부터 정해져 있었다.
```

즉

```
숨은 변수 존재
```

---

# 2. 양자 얽힘과의 차이

양자 얽힘에서는

```
측정 방향
```

을 선택할 수 있다.

예

```
측정 방향
A
B
C
```

Alice와 Bob이 각각 방향을 고른다.

---

# 3. 숨은 변수 세계의 예측

만약 양말 상자처럼

```
결과가 미리 정해져 있다면
```

입자는 사실 이렇게 정보를 가지고 있어야 한다.

예

```
A방향 → ↑
B방향 → ↓
C방향 → ↑
```

즉 입자는

```
모든 방향에 대한 결과 목록
```

을 이미 가지고 있다.

---

# 4. 벨의 핵심 아이디어

이렇게

```
미리 정해진 결과 목록
```

이 존재한다면

Alice와 Bob의 결과 사이에는

```
상관관계 최대값
```

이 생긴다.

그 최대값이 바로

```
벨의 부등식
```

이다.

즉

```
숨은 변수 세계가 만들 수 있는
상관관계의 한계
```

이다.

---

# 5. 실제 실험

얽힌 입자를 측정하면

```
실험 상관관계
>
벨 부등식 한계
```

가 나타난다.

즉

```
양말 상자 모델
```

같은 설명으로는

```
재현 불가능
```

---

# 6. 직관적 의미

양말 상자 모델

```
결과 = 이미 존재
```

양자 얽힘

가능성

```
1. 측정 전 결과 없음
2. 비국소 연결 존재
3. 측정 선택도 미리 결정
```

---

## 한 줄 결론

벨의 부등식은

```
“우주의 상관관계는
양말 상자 같은
미리 정해진 값 모델로
설명될 수 없다”
```

는 것을 보여준다.

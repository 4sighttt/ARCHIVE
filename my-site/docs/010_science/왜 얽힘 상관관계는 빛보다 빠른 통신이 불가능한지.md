## 핵심 요약

- 양자 **얽힘(entanglement)**은 두 입자의 결과가 강하게 **상관**되게 만든다.
    
- 그러나 각 측정 결과는 **개별적으로 보면 완전히 무작위**다.
    
- 한쪽에서 측정 방식을 바꿔도 **상대방이 보는 통계는 변하지 않는다.**
    
- 따라서 상대방은 **그 변화가 있었는지 알 방법이 없다.**
    
- 정보를 전달하려면 **의도적으로 바꿀 수 있는 신호**가 필요하지만 얽힘 결과는 통제 불가능하다.
    
- 그래서 얽힘은 **초광속 상관관계는 있지만 초광속 통신은 없다.**
    

---

# 1. 얽힘 실험의 기본 구조

얽힌 두 입자 생성

```
source → A(왼쪽) + B(오른쪽)
```

멀리 떨어진 두 실험자가 측정

```
Alice  (A 측정)
Bob    (B 측정)
```

---

# 2. 결과는 항상 무작위

예: 스핀 측정

Alice 결과

```
↑ 또는 ↓
확률 50%
```

Bob 결과

```
↑ 또는 ↓
확률 50%
```

각자 **단독으로 보면 완전한 랜덤**이다.

---

# 3. 상관관계는 존재한다

두 사람이 결과를 나중에 비교하면

예

```
Alice ↑  → Bob ↓
Alice ↓  → Bob ↑
```

같은 방향 측정 시 항상 반대.

즉

```
각각은 랜덤
하지만 둘은 강하게 연결
```

---

# 4. 왜 이것으로 통신이 안 되는가

Alice가 Bob에게 메시지를 보내고 싶다고 가정.

예

```
0 → 측정하지 않음
1 → 측정함
```

하지만 Bob이 보는 결과:

```
↑ ↓ ↑ ↓ ↓ ↑ ↓ ↑ ...
```

여전히

```
50% / 50%
```

Alice가 무엇을 했는지 **통계적으로 구별 불가능**.

즉

```
Bob의 데이터
= 항상 랜덤
```

---

# 5. 핵심 이유

얽힘의 특징

```
상관관계는 존재
하지만 결과는 제어 불가능
```

정보 전달에는

```
제어 가능한 신호
```

가 필요하다.

하지만 양자 측정은

```
결과 조작 불가능
```

---

# 6. 물리학에서의 결론

이를 보통

**No-communication theorem**

이라고 부른다.

요약

```
얽힘 → 초광속 상관관계
하지만

정보 전달 → 불가능
```

그래서

```
양자역학
+
상대성이론
```

이 **충돌하지 않는다.**

---

원하면 다음도 설명할 수 있다.

- 왜 **벨의 부등식 위반은 고전 확률로 절대 재현 불가능한지**
    
- 얽힘이 **양자컴퓨터와 양자암호에서 어떻게 쓰이는지**
    
- 아인슈타인이 말한 **“spooky action at a distance”**의 정확한 의미.

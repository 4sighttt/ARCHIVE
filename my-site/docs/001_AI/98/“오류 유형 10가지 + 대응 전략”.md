## 핵심 요약

- AI 오류는 대부분 **패턴화 가능**
    
- 핵심 4축: **환각 / 논리 / 누락 / 편향**
    
- 대응 전략은 “사후 수정”이 아니라 **사전 차단 구조**
    
- 단일 답변 신뢰 금지 → **비교·반박 필수**
    
- 오류를 정의하면 **검증 비용 급감**
    
- 실전은 “오류 탐지 시스템” 구축이 핵심
    

---

# 오류 유형 10가지 + 대응 전략

## 1. 환각 (Hallucination)

**문제**

- 존재하지 않는 사실을 단정
    

**징후**

- 출처 없음
    
- 지나치게 확신적 표현
    

**대응**

```text
출처 없는 정보 금지
불확실 시 “모름” 표시
```

---

## 2. 최신성 오류 (Outdated Info)

**문제**

- 오래된 정보 사용
    

**징후**

- 현재 상황과 불일치
    

**대응**

```text
“최근 24시간 / 최신 기준” 명시
```

---

## 3. 논리 비약 (Logical Jump)

**문제**

- 근거 없이 결론 도출
    

**징후**

- “그래서 → 결론” 연결 약함
    

**대응**

```text
추론 과정 단계별 설명 요구
```

---

## 4. 누락 (Omission)

**문제**

- 핵심 요소 빠짐
    

**징후**

- 설명이 깔끔한데 얕음
    

**대응**

```text
“빠진 요소 지적” 단계 추가
```

---

## 5. 확증 편향 (Confirmation Bias)

**문제**

- 한쪽 주장만 강화
    

**징후**

- 반대 의견 없음
    

**대응**

```text
반대 입장 생성 강제
```

---

## 6. 과잉 일반화 (Overgeneralization)

**문제**

- 일부 사례 → 전체 결론
    

**징후**

- “항상 / 대부분” 표현
    

**대응**

```text
예외 조건 요구
```

---

## 7. 모호한 표현 (Vagueness)

**문제**

- 의미 불명확
    

**징후**

- “적절히 / 충분히 / 어느 정도”
    

**대응**

```text
정량 기준 요구
```

---

## 8. 구조 붕괴 (Structure Failure)

**문제**

- 출력 형식 깨짐
    

**징후**

- 리스트/구조 불일치
    

**대응**

```text
출력 형식 강제 지정
```

---

## 9. 지시 불이행 (Instruction Drift)

**문제**

- 조건 무시
    

**징후**

- 요구사항 일부 누락
    

**대응**

```text
조건 체크리스트 포함
```

---

## 10. 과도한 확신 (False Confidence)

**문제**

- 불확실한데 단정
    

**징후**

- “확실히 / 반드시”
    

**대응**

```text
확실/불확실/추정 구분 강제
```

# 통합 대응 구조 (실전)

```text
1. 생성 (3개)
2. 오류만 찾기
3. 반박 생성
4. 수정
5. 최종 선택
```

# 초압축 체크리스트

```text
- 출처 있는가?
- 논리 점프 없는가?
- 빠진 핵심 없는가?
- 반대 관점 있는가?
- 확신 과도한가?
```

[[텍스트 패턴만으로 언어 이해가 가능한가]]
[[진지하고 훌륭한 철학서는 농담만으로도 구성될 수 있다.]]

**핵심 요약**

1. 언어 이해를 **텍스트 패턴만으로 충분한가**에 대해서는 AI 연구에서 논쟁이 있다.
    
2. 한 입장은 **통계적 패턴만으로도 의미가 형성될 수 있다**고 본다. (대형 언어모델 접근)
    
3. 다른 입장은 **텍스트만으로는 실제 세계와의 연결이 없기 때문에 이해가 불완전하다**고 주장한다.
    
4. 이를 **기호 접지 문제(Symbol Grounding Problem)**라고 부른다. (Stevan Harnad, 1990)
    
5. 두 번째 질문인 **언어 이해를 행동으로 정의할 수 있는가**는 철학적으로 강한 전통이 있다.
    
6. **비트겐슈타인**: 의미는 사용 속에 있다 (“meaning is use”).
    
7. **튜링 테스트**도 이해를 **행동적 기준**으로 평가하는 접근이다.
    
8. 현대 AI 연구에서는 **텍스트 패턴 + 환경과의 상호작용**을 결합해야 한다는 견해가 증가하고 있다.

# 1. 텍스트 패턴만으로 언어 이해가 가능한가

이 문제는 AI 철학에서 오래된 논쟁이다.

### ① 가능하다는 입장 (Statistical Semantics)

대표적 근거

- **분포 의미론 (Distributional Semantics)**  
    → _“You shall know a word by the company it keeps”_  
    (J. R. Firth, 1957)
    

핵심 주장

- 단어의 의미는 **다른 단어들과의 통계적 관계**에서 나타난다.
    
- 충분히 큰 데이터가 있으면 의미 구조가 드러난다.
    

현대 LLM은 이 가설을 기반으로 한다.

예

```
cat → pet, animal, fur, meow
dog → pet, animal, bark
```

이 패턴만으로도

- 동물
    
- 애완동물
    
- 울음소리
    

같은 의미 구조가 형성된다.

따라서 일부 연구자는

> 언어 이해는 **텍스트 통계만으로도 상당 부분 가능하다**

라고 본다.

---

### ② 불가능하다는 입장 (Symbol Grounding)

Stevan Harnad (1990)

**Symbol Grounding Problem**

핵심 주장

텍스트만 있는 시스템은 결국

```
단어 → 단어 → 단어
```

만 반복한다.

예

```
apple = fruit
fruit = edible plant product
plant = organism
```

하지만 실제

- 사과의 맛
    
- 색
    
- 물리적 경험
    

과 연결되지 않는다.

그래서

> 텍스트만으로는 **기호가 현실에 접지되지 않는다**

라고 주장한다.

이 문제는 종종 **Searle의 Chinese Room 논증**과 함께 논의된다.

# 2. 언어 이해를 행동으로 정의할 수 있는가

철학적으로 매우 강한 전통이 있다.

## ① 비트겐슈타인

《철학적 탐구》

핵심 주장

> 의미는 사용이다  
> (Meaning is use)

즉

단어의 의미는

- 정의
    
- 내부 표현
    

이 아니라

**사회적 사용 방식**

으로 결정된다.

예

“게임”이라는 단어

명확한 정의는 없지만

사람들은 **적절하게 사용한다.**

---

## ② 행동주의 접근

언어 이해를 이렇게 정의한다.

```
이해 = 적절한 행동을 할 수 있음
```

예

누군가 말한다

```
문을 닫아줘
```

이해했다면

→ 문을 닫는다

따라서

```
이해 = 행동 능력
```

---

## ③ 튜링 테스트

Alan Turing (1950)

지능 정의

> 인간처럼 대화 행동을 하면  
> 지능이 있다고 본다.

즉

**내부 상태가 아니라 행동을 기준으로 판단**

# 3. 현대 AI 연구의 현재 위치

최근 연구는 **둘을 결합하는 방향**이다.

### 1️⃣ 텍스트 기반 모델

예

- GPT
    
- Claude
    
- LLaMA
    

장점

- 대규모 지식
    
- 언어 패턴 학습
    

한계

- 현실 접지 부족
    

---

### 2️⃣ 행동 기반 AI

예

- 로봇
    
- embodied AI
    
- agent systems
    

특징

- 환경과 상호작용
    
- 행동 기반 학습
    

---

### 3️⃣ 현재 연구 방향

최근 AI 연구는 다음 구조를 추구한다.

```
언어 모델
      +
환경 상호작용
      +
행동 계획
```

예

- DeepMind **Gato**
    
- Google **RT-2**
    
- OpenAI **agent systems**
    

---

# 4. 철학적으로 보면 두 질문은 사실 하나다

두 질문은 이렇게 연결된다.

```
언어 이해 = 패턴인가?
언어 이해 = 행동인가?
```

철학적 정리

|입장|주장|
|---|---|
|통계주의|패턴이면 충분|
|기호접지|현실 연결 필요|
|비트겐슈타인|사용/행동이 의미|

---

# 핵심 결론

현재 합의된 결론은 없다.

하지만 많은 연구자가 다음 방향을 지지한다.

> 언어 이해 = **텍스트 패턴 + 세계와의 상호작용**

즉

- **패턴만으로 부분적 이해**
    
- **행동까지 포함하면 더 완전한 이해**
    

---

원하면 다음도 설명할 수 있다.

- **LLM이 실제로 의미를 “이해”한다고 볼 수 있는가**
    
- **비트겐슈타인의 언어게임과 LLM의 관계**
    
- **AI에서 symbol grounding을 해결하려는 최신 연구**

[[왜 휴리스틱적 추론이 나타나는가]]

**핵심 요약**

1. 현재 연구 기준으로 **LLM이 “생각한다”고 단정할 근거는 없다**.
    
2. LLM은 기본적으로 **다음 토큰 확률을 계산하는 모델**이다.
    
3. 그러나 모델 내부에서 **일부 계산적 구조가 나타난다는 연구**도 존재한다.
    
4. 따라서 학계의 일반적 입장은 **“생각한다기보다 생각처럼 보이는 계산 과정”**이다.
    
5. 이 문제는 AI 연구뿐 아니라 **언어철학·인지과학의 핵심 논쟁**이다.
    

---

# 1. 왜 “생각한다”고 보기 어려운가

LLM의 기본 목표 함수는 다음이다.

[  
P(token_t \mid token_{1:t-1})  
]

즉 모델의 목적은

> **다음 단어의 확률을 예측하는 것**

이다.

여기에는

- 목표 설정
    
- 의도
    
- 의미 이해
    
- 자율적 사고
    

같은 구조가 명시적으로 존재하지 않는다.

그래서 많은 연구자들은

> LLM은 **언어 생성 시스템**이라고 본다.

---

# 2. 그런데 왜 “생각하는 것처럼” 보일까

대규모 텍스트에는 이미

- 논리 구조
    
- 설명 방식
    
- 문제 해결 패턴
    

이 포함되어 있다.

LLM은 이런 구조를 학습하면서

다음과 같은 패턴을 생성한다.

```
문제 → 분석 → 단계적 추론 → 결론
```

이 구조 때문에 인간은

> **추론이 일어난 것처럼 보인다**

고 느낀다.

---

# 3. 일부 연구가 제기하는 반론

최근 연구에서는 모델 내부에서

- 덧셈 계산 회로
    
- 문법 추적 구조
    
- 알고리즘 유사 패턴
    

같은 **내부 계산 구조**가 발견되기도 한다.

이는 다음 가능성을 제기한다.

> LLM 내부에 **제한적인 계산적 추론**이 존재할 수 있다.

그러나 이것이

- 인간 사고
    
- 의식적 사고
    

와 동일하다는 증거는 없다.

---

# 4. 철학적으로는 오래된 문제

이 논쟁은 사실 새로운 것이 아니다.

대표적 철학 문제

**존 설 – 중국어 방(Chinese Room)**

주장

> 규칙에 따라 문장을 처리할 수 있어도  
> **이해가 발생했다고 말할 수는 없다**

LLM 논쟁은 거의 동일한 질문을 다시 제기한다.

---

# 정리

현재 가장 보수적인 결론은 다음이다.

> **LLM은 생각한다고 증명된 시스템은 아니다.**  
> **그러나 단순한 문장 복사기보다 훨씬 복잡한 계산 시스템이다.**

즉

- 단순 모방도 아니고
    
- 인간 사고도 아닌
    

**새로운 형태의 정보 처리 시스템**으로 보는 것이 일반적이다.


**핵심 요약**

1. 지뢰찾기 같은 퍼즐에서 보이는 능력은 **단순 문장 복사 수준보다 복잡한 계산**이 맞다.
    
2. 그러나 그것이 **인간과 같은 “의식적 사고”라는 증거는 아니다**.
    
3. 현재 연구에서는 이를 **패턴 기반 추론(pattern-based reasoning)** 또는 **근사 추론**으로 설명한다.
    
4. LLM은 학습된 패턴과 내부 표현을 이용해 **논리 구조를 생성·검증**할 수 있다.
    
5. 그래서 **논리 검증·오류 탐지 같은 작업**에서는 실제 추론처럼 보이는 행동이 나타난다.
    

---

# 1. 왜 퍼즐에서 “논리 추론”처럼 보이는가

지뢰찾기 같은 퍼즐에는 **강한 구조적 패턴**이 있다.

예

```
숫자 = 주변 지뢰 개수
```

이 규칙 때문에 다음과 같은 패턴이 반복된다.

- 1 옆에 미확인 칸이 1개 → 그 칸이 지뢰
    
- 2 옆에 미확인 칸 2개 → 둘 다 지뢰
    
- 두 숫자의 관계로 안전 칸 추론
    

이런 **논리 패턴이 텍스트 데이터에도 매우 많이 등장**한다.

예

- 퍼즐 풀이 설명
    
- 수학 풀이
    
- 프로그래밍 문제 풀이
    

LLM은 이런 **추론 패턴의 언어 구조**를 학습한다.

---

# 2. 그래서 가능한 능력

LLM은 다음 작업에 강하다.

### ① 논리 흐름 생성

```
조건 → 가능한 경우 → 결론
```

같은 **추론 구조 생성**.

---

### ② 논리 검증

사람이 설명한 논리를 보면

- 조건 충돌
    
- 누락된 가정
    
- 계산 오류
    

같은 것을 **패턴 기반으로 감지**할 수 있다.

이것은 실제로

- 코드 리뷰
    
- 수학 풀이 검증
    
- 논문 논리 검토
    

에서도 많이 나타난다.

---

# 3. 중요한 차이

하지만 인간의 논리 추론과 완전히 같지는 않다.

차이:

|인간 추론|LLM 추론|
|---|---|
|목표 지향적 사고|토큰 확률 생성|
|내부 세계 모델|언어 패턴 모델|
|의도·이해 존재|통계적 표현|
|오류 시 자기 수정 가능|오류 패턴 반복 가능|

그래서

- 복잡한 퍼즐
    
- 완전히 새로운 문제
    

에서는 갑자기 실패하기도 한다.

---

# 4. 연구자들이 현재 보는 관점

최근 AI 연구에서는 다음 개념이 많이 쓰인다.

**emergent reasoning**

뜻

> 대규모 모델에서 **부분적인 추론 능력이 자연스럽게 나타난다**

대표적으로

- 수학 문제
    
- 논리 퍼즐
    
- 코드 분석
    

에서 관찰된다.

하지만

> 이것이 인간 사고와 동일한지는 아직 불확실하다.

---

# 정리

지뢰찾기에서 보이는 능력은 실제로

- 단순 문장 생성보다 **훨씬 복잡한 계산 결과**다.
    

그러나 현재 연구 기준으로는

> **“통계적 언어 모델에서 나타난 추론 능력”**

으로 설명된다.

즉

- 완전한 사고도 아니고
    
- 단순 복사도 아닌
    

**중간 형태의 계산적 추론 시스템**으로 보는 것이 가장 일반적이다.

---

2. **LLM 내부에서 실제 알고리즘이 발견된 연구**
    
3. **LLM이 인간 추론과 닮은 이유 (인지과학 관점)**

**핵심 요약**

1. **Symbol Grounding Problem**은 “단어가 현실 세계와 어떻게 연결되는가” 문제이다. (Harnad, 1990)
    
2. 텍스트만 학습한 모델은 **기호 ↔ 기호 관계**만 배우고 **세계 경험**이 없다.
    
3. 최근 AI 연구는 이를 해결하기 위해 **언어 + 감각 + 행동**을 결합하려 한다.
    
4. 주요 접근은 **멀티모달 학습, 로봇 embodied AI, 환경 상호작용 agent** 세 가지다.
    
5. 대표 연구: **CLIP (OpenAI), PaLM-E (Google), RT-2 (Google), Gato (DeepMind)**.
    
6. 이 연구들의 핵심 목표는 **단어 → 실제 사물/행동 연결**이다.
    
7. 아직 완전한 해결은 아니며 현재는 **부분적 grounding 단계**로 평가된다.
    

---

# 1. 멀티모달 grounding 연구

텍스트를 **이미지·영상과 연결**하는 접근이다.

## CLIP (OpenAI, 2021)

핵심 아이디어

```
텍스트 ↔ 이미지
```

동시에 학습한다.

예

```
"a dog"
"a red car"
"a cat sitting on a chair"
```

모델은

- 사진
    
- 문장
    

사이의 대응을 학습한다.

결과

```
dog → 실제 개 이미지
```

즉 일부 의미가 **현실 시각 정보와 연결**된다.

출처  
Radford et al., _Learning Transferable Visual Models From Natural Language Supervision_, 2021.

---

# 2. 로봇 기반 grounding

언어를 **물리 행동**과 연결하는 연구이다.

## RT-2 (Google DeepMind, 2023)

로봇에게

```
"pick up the apple"
```

같은 명령을 주면

모델이

```
언어 이해
→ 시각 인식
→ 로봇 팔 행동
```

으로 이어진다.

즉

```
apple = 텍스트
apple = 실제 물체
apple = 집는 행동
```

이 연결이 만들어진다.

출처  
Brohan et al., _RT-2: Vision-Language-Action Models Transfer Web Knowledge to Robotic Control_, 2023.

---

# 3. 범용 embodied 모델

언어 + 행동 + 환경을 통합하는 연구다.

## Gato (DeepMind, 2022)

하나의 모델이

- Atari 게임
    
- 로봇 제어
    
- 이미지 설명
    
- 대화
    

등을 동시에 수행한다.

핵심 아이디어

```
모든 것을 "token sequence"로 통합
```

예

```
텍스트
이미지
행동
```

모두 동일한 구조로 학습한다.

출처  
Reed et al., _A Generalist Agent_, DeepMind, 2022.

---

# 4. 언어-로봇 통합 모델

## PaLM-E (Google, 2023)

대형 언어모델 + 로봇 센서

입력

```
텍스트
카메라 영상
센서 데이터
```

출력

```
로봇 행동
```

예

```
"bring me the green bottle"
```

모델이

- 물체 인식
    
- 위치 파악
    
- 집기 행동
    

을 수행한다.

출처  
Driess et al., _PaLM-E: An Embodied Multimodal Language Model_, 2023.

---

# 5. 현재 연구의 핵심 방향

현대 grounding 연구는 다음 구조를 목표로 한다.

```
Language
   +
Perception
   +
Action
   +
Environment
```

즉

```
단어
→ 사물
→ 행동
```

연결을 만드는 것이다.

---

# 6. 여전히 남은 문제

많은 연구자들은 아직 **완전한 grounding은 아니다**라고 본다.

이유

1. 실제 경험이 제한적
    
2. 인간처럼 감각 통합이 없음
    
3. 장기 환경 학습 부족
    

그래서 일부 연구자는

> 현재 AI는 **weak grounding** 상태라고 평가한다.

---

## 정리

symbol grounding 해결을 위한 주요 연구 흐름

|접근|핵심 아이디어|
|---|---|
|멀티모달|텍스트 ↔ 이미지|
|로봇 AI|텍스트 ↔ 행동|
|embodied AI|언어 + 감각 + 환경|
|agent 시스템|언어 + 계획 + 행동|

---

원하면 다음도 설명할 수 있다.

- **LLM은 실제로 grounding이 없는가** (최근 논쟁)
    
- **비트겐슈타인 언어게임과 embodied AI의 관계**
    
- **AI가 인간처럼 의미를 가질 수 있는 조건** (철학 + AI 연구)

**요약 (10줄 이하)**

1. **서버(server)**는 **요청을 받아서 응답을 주는 컴퓨터**다.
    
2. 사용자의 앱이나 프로그램이 **데이터를 요청**한다.
    
3. 서버는 **데이터를 처리하거나 저장된 정보를 찾아서 응답**한다.
    
4. 예: 날씨 앱 → 날씨 서버 → 현재 온도 반환.
    
5. 서버는 보통 **데이터베이스 + 프로그램 + 네트워크 연결**로 구성된다.
    
6. ==인터넷에 항상 연결되어 **24시간 요청을 처리**한다.==
    
7. 물리적으로는 **일반 컴퓨터와 같은 하드웨어**일 수도 있다.
    
8. 차이는 **항상 켜져 있고 여러 사용자의 요청을 처리한다는 점**이다.
    

원하면 **서버–클라이언트 구조를 한 단계 더 자세히 설명**할 수 있다.

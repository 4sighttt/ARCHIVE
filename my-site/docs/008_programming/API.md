**요약 (10줄 이하)**

1. ==**API (Application Programming Interface)**는 **프로그램끼리 대화하는 규칙**이다==.
    
2. 한 프로그램이 **다른 프로그램의 기능을 요청하는 방법**을 정의한다.
    
3. 예: 날씨 앱 → 날씨 서버 API → 온도 데이터 받음.
    
4. 개발자는 **내부 구조를 몰라도 기능을 사용할 수 있다.**
    
5. 보통 **요청(request) → 응답(response)** 구조.
    
6. 인터넷에서는 **HTTP API / REST API** 형태가 많다.
    
7. 예: 지도, 결제, 로그인, AI 기능 등이 API로 제공된다.
    
8. 대표 사례: **OpenAI API**
    

원하면 **API가 실제로 어떻게 호출되는지(구조 예시)** 한 단계 설명할 수 있다.
